# Hacktoberfest 是什么？
__有问题？这里是关于 Hacktoberfest 的常见问题及解答。__

## 一般原则
### Hacktoberfest 是什么？
Hacktoberfest is a month-long celebration of open source software run by DigitalOcean in partnership with GitHub and Twilio.
Hacktoberfest 是由 DigitalOcean 联合 Github 和 Twilio一起发起的、持续一个月的一场开源软件庆祝活动。

* Hacktoberfest 面向全球社区的所有人开放！
* 必须要提交5个合格的PR到公开的 Github 仓库。
* 可以在10月1号到31号期间的任意时间注册参与。

### 如何开始？
在 https://hacktoberfest.digitalocean.com/ 注册并提交PR到任何 Github 公开仓库。鼓励高质量的贡献！可以寻找带有 “Hacktoberfest” 标签的问题来获取灵感。

有维护一个仓库？

在你的 Github 项目中创建问题标签，或者为已有问题打上 “Hacktoberfest” 标签，已帮助新的贡献者了解有该做什么。为垃圾信息或无关的 PR 打上 “invalid” 标签让使其不合格。

### 如何跟踪我的进度？
使用你的 GitHub 账户登录到 https://hacktoberfest.digitalocean.com/ 来查看你的进度和统计数据。

### 我完成了5个PR，什么时候可以收到T恤？
我们会在月内开始发送关于T恤的详细信息的邮件。敬请期待！

### 如何获得贴纸？
贴纸与T恤是一起的，只要你完成5个PR。

### 如果我到月底完成的PR数量少于5个怎么办？
很不幸你必须要提交至少5个PR才能获得T恤。

## 规则
### 可以使用非 Github 账户来贡献吗？
不行，必须在 Github 平台上提交PR。

### 问题或提交信息会算在数量内吗？
不会，只有PR算数。

### 我在10月中旬注册了 Hacktoberfest ，那么我在此前的10月份里创建的PR算数吗？
算，所有在10月1号到31号期间提交的PR都算数，不管你合适注册 Hacktoberfest 。在10月1号前创建的PR但是已被合并的不算数。

### 必须是带有 #Hacktoberfest 标签的问题才计数吗？
不是，任何提交到公开仓库的PR都算数，不管其关联的问题是否到有 #Hacktoberfest 标签。

### 提交到我自己仓库的PR算数吗？
算，但是我们强烈建议你贡献高质量的PR到其他仓库。

### 可以提交PR 到未在主页列出的问题或仓库里吗？
可以，只要仓库被批准。

### 提交多个PR到同一个仓库计为多个PR数吗？
是的，每一个都会被单独计数。

### 提交的PR一定要被接受或合并吗？
不是。只要它们没有被标记非垃圾信息、无关或打上不合格标签，都会被计数。

## 邮寄
### 收到T恤/贴纸时需要支付什么费用吗？
T恤对参与者是免费的，我们也会承担邮寄的费用。

### 我没有收到2017年 Hacktoberfest 的T恤/纪念品！
我们不再印制2017年的T恤。如果你没有收到T恤，意味着我们已经尝试过多次但依然无法搞定。我们正努力改进今年的快递流程，以让大家都可以收到自己的T恤，感谢理解。

### 需要为T恤支付任何费用/消费税/关税之类的吗？
我们在邮寄包裹上写下了一个美元价格，预计足够低不会产生关税，但并不能保证这一点 —— 根据你所在国家的进口政策也有可能需要支付一点儿税费。

### T恤什么时候发货？（国际）
一旦T恤发货你就会收到一封邮件。国际订单需要在发货3到5周后送达。

### T恤什么时候发货？（美国）
一旦T恤发货你就会收到一封邮件。美国境内订单需要在发货1到2周后送达。 

### 如何退货或换货？
如果你确信自己的订单漏发了，请在付款的30天内，通过在邮件标题中提供订单号和所收错误包裹解释来联络我们，参考指南 https://kotisdesign.com/returns 。你需要承担寄回给我们的快递费用。

### 我们的国际订单延迟了，怎么办？
我们推荐联系你所在国家的DHL，来获取详细的包裹信息： http://www.dhl.com/en/contact_center/contact_parcel_ecommerce.html
